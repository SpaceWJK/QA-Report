  • 에픽세븐 구버전: EP7
  • 카제나 버그: GCZ (~2,981건)
  • 카제나 운영/빌드: CBOCZN
  • 로드나인 버그: LDN (~3,849건)
  • 로드나인 아시아: LNA
이슈 제목 패턴
  • [QA X.X.X] QA 환경 발견 버그

  • [카제나][빌드][0617] 빌드 일정
  • 우선순위: Blocker > Critical > Major > Minor
  • 상태: TE Review → 진행중 → Resolve → 닫힘

/gdi

GDI — 기획서(xlsx·pptx)·게임 데이터 테이블(tsv)

게임별 커버리지
  • 카오스제로나이트메어: xlsx·pptx·TSV 전부 ✅
  • 에픽세븐: TSV 게임 데이터만 ⚠️
  • 로드나인: 미인덱싱 ❌
  • 로드나인 아시아: 미인덱싱 ❌
파일 종류
  • xlsx: 패치노트·캐릭터 사양서·QA 사양서·밸런스
  • pptx: 기능 기획서·UI 플로우·작업 근거 문서
  • tsv: 스킬·스테이지·드랍·미션·재화 원본 테이블
💡 카제나 파일 명명 규칙 — #이슈번호 작업자이름 - 작업내용.xlsx 형식. 캐릭터명·작업자명·이슈번호로 검색 가능.
✅ 활용 예시
/gdi 카제나 6월 17일 타겟 업데이트의 주요 기획서 항목이나 업데이트될 컨텐츠 및 기능들 정리해줘
/gdi 카제나 나인 캐릭터 사양서 내용 알려줘
/gdi 카제나 최신 패치노트 주요 변경사항
/gdi 카제나 시즌 2 전투원 밸런스 수정 리스트
/gdi 카제나 포텐셜 노드 데이터 테이블
/gdi 에픽세븐 드랍 테이블 구조
/gdi 카제나 튜토리얼 관련 기획서 있어?
❌ 이런 질문은 다른 명령어로
/gdi 로드나인 기획서→ 미인덱싱. /wiki 또는 /jira 사용
/gdi 에픽세븐 패치노트 기획서→ 에픽세븐 기획 문서 없음. TSV만 가능
/biskit — 실시간 지표 · 커뮤니티 감성
동접, DAU, 리텐션, 커뮤니티 긍부정 감성 분석 데이터를 조회합니다.

/biskit

BISKIT — 실시간 동접·DAU·리텐션·마케팅 ROAS·커뮤니티 감성

공통 지표 (4개 게임)
  • 실시간 동접 (CCU) — 5분 단위
  • 커뮤니티 긍/중/부정 비율 (일별)
  • 커뮤니티 키워드 랭킹·순위 변동
  • LLM 자동 여론 요약
  • 기간 비교 분석 (전주 대비 등)
게임별 특화 지표
  • 에픽세븐: DAU·마케팅 ROAS (15개국)
  • 카제나: 리텐션 D1~D28·크리스탈 경제·콘텐츠 플레이
  • 로드나인: CCU + 커뮤니티만
  • 로드나인 아시아: CCU + 사전등록 국가별
⚠️ 매출·ARPU 지표 조회 불가 — 현재 권한 범위에 포함되지 않습니다. 매출 관련 질문은 담당자에게 문의하세요.
✅ 활용 예시
/biskit 카제나 6월 1주차 커뮤니티 유저들의 긍정, 부정, 중립 동향 비율 알려줘

/biskit 카제나 D7 리텐션 최근 추이 보여줘
/biskit 에픽세븐 이번 달 한국 캠페인 D7 ROAS
/biskit 카제나 크리스탈 유료/무료 소모 현황
/biskit 로드나인 아시아 국가별 사전 등록자 수
